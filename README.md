    # 🎓 LMS Portal — Система управління навчальним процесом

Веб-платформа для автоматизації шкільного життя: керування вчителями, предметами, учнями та оцінюванням. Проєкт розроблений у межах командного турніру з програмування.

## ✨ Основні можливості
- **Авторизація:** Надійна система входу з використанням JWT-токенів.
- **Дашборд:** Швидка статистика (кількість учнів, класів та предметів).
- **Керування штатом:** Списки викладачів та учнів із пошуком.
- **Класний журнал:** Зручна таблиця для виставлення оцінок (1-12) та обліку відвідуваності.
- **Розклад занять:** Календарне відображення уроків на тиждень.
- **Сучасний UI:** "Glassmorphism" дизайн, реалізований на Tailwind CSS 4.

---

## 🛠 Технологічний стек
- **Backend:** Node.js, Express.js, TypeScript, Prisma ORM, SQLite, JWT, Bcrypt.
- **Frontend:** React 19, Vite, React Router, Tailwind CSS 4, Lucide React.
